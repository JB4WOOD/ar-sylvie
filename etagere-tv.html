Étagère TV - Projet Sylvie

Comment tester dans votre pièce ?

  • Tournez le modèle avec le doigt pour le voir sous tous les angles
  • Pincez pour zoomer / dézoomer
  • Appuyez sur "Voir dans ma pièce" pour placer l'étagère au-dessus de votre TV via la caméra
  • Déplacez l'étagère en la faisant glisser sur le sol/mur
